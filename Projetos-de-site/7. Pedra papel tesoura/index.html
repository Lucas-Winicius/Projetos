Pedra Papel Tesoura

Jogo pedra papel tesoura criado por Lucas Winicius para diversão pessoal e treinamento de HTML CSS e JavaScript

Você pode jogar sozinho ou com um amigo

Regras

  • Tenha sorte
  • Faça 10 pontos primeiro
  • Se divirta

Como jogar

Para jogar selecione um numero de 0 a 2

  • 0 Indica pedra
  • 1 Indica papel
  • 2 Indica tesoura

Identificações respectivas

Modos de Jogo

  • PvP: Jogador vs Jogador quem sera o melhor. Neste modo você pode jogar contra seu amigo.
  • PvE: Jogador vs Maquina jogo limpo e sem trapaças. Neste modo sera apenas você contra sua Maquina
